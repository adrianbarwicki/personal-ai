# Languages

Personal language-learning notes — vocabulary, cultural cheat sheets, and situational scripts.

## Current focus

- **Russian** — primary working language for meeting Lia's parents (post-Soviet Belarusian family, Russian-speaking at home)
- **Belarusian** — secondary; a few well-chosen phrases land hard if parents value Belarusian identity

## Structure

| Folder | Purpose |
|--------|---------|
| `Russian/` | Russian vocabulary, scripts, cultural notes |
| `Belarusian/` | Belarusian equivalents — use selectively, see notes |

## How to use

These are reference docs, not lessons. Open the relevant file before a specific event (dinner, first meeting, phone call), drill the phrases that match the scenario, and go.

Files are organized by **situation** rather than by grammar — easier to find what you actually need to say.
